datafaker

datafaker is a package for making copies of relational databases and populating them with random data.

If you are new to datafaker, we recommend going through the pages of this documentation roughly in order: After installing datafaker and learning the basic commands that it uses from the quick start guide, the introductory tutorial will walk you through a relatively simple use case in detail. You can then look at one of our other two example use cases, one for financial data and one for health data. The latter also goes through some more advanced features of datafaker and how to use them, that are relevant beyond health data use cases.

Note

This project will be under active development from Jan - Oct 2023
